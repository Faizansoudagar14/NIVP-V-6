  • Customer Restricted Usage in World Wide?
    Promotions that are valid only 'xx' times per customer or per account or per order where 'xx' can be any number equal to or greater than 1 will be matched by populating the exact value of ‘xx’ in Maximum Restricted No of Usage field that is available in Vibe and Myra forms of UPT. For example, if promotion is valid once per customer/order/account, the Maximum Restricted No of Usage = 1 and if it is valid 5x per customer/order/account Maximum Restricted No of Usage = 5.

    If rebate is less than 10%, overall points on DP will be base points
    If rebate is greater than 10%, 10% rounded up will be base and the remaining as NBB points.

  • What is Basket level promotion?
    To avail a promotion if there is a certain price threshold basket which needs to be built only after which rebate is applicable, then such promotions are called as Basket level promotions. For ex: Spend $50 and get $10 off, Extra 20% off when you spend $100 or more.
  • What is multibuy promotion?
    To avail a promotion if more than a single product needs to be bought/purchased together then such promotions are called as multibuy promotions. For ex: Buy 5 items, get 10% off, Buy 3 for the price of 2, Buy 6 items, get $20 off.
  • Can we match Members only promotion in World Wide?
    Yes, only if a promotion is ‘Free of cost’(without friction i.e., if the membership can be obtained just by signing in the competitor) we can match such promotions. But only in FR MKPL - Please match All(SICs/ICs) Member only promotions in Y type even if it is Free or requires Payment for membership.

  • Pickup In Store:
    Promotions that let customers place orders online and visit a store to pick up the order will be matched by us by using corresponding rebate type (we will not use “Pickup_In_store” as rebate type). For example, if a promo is 10% off, pickup in store, we will select rebate type as Straight NBB (not Pickup_In_store).
